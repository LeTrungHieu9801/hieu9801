Tìm hiểu chung về HTML - Code thiết kế web bằng HTML

1. HTML là gì?

HTML (Hypertext Markup Language) là một loại ngôn ngữ đánh dấu siêu văn bản, được sử dụng để bố cục các thành phần có trên web, định dạng web, phân chia đoạn văn, heading, link,…

Vai trò chính của HTML

+ Dùng để xây dựng cấu trúc nội dung của web/ ứng dụng web
+ Khai báo, định dạng các tập tin kỹ thuật số
+ Tạo style cơ bản nhất cho website, có thể kết hợp cùng CSS để tăng tính thẩm mỹ cho web (màu sắc, kiểu backgrounds, bố cục, hiệu ứng,…), kết hợp với JavaScript để thêm các chức năng động khác cho web (thư viện ảnh/ slider/ pop-up).

HTML là gì?

2. Ưu, nhược điểm của HTML

Ưu điểm

- Dễ hiểu, dễ đọc
- Hoàn toàn miễn phí
- Sử dụng được trên tất cả các trình duyệt web (Chrome, Cốc cốc, FireFox…)
- Nguồn tài nguyên hỗ trợ dồi dào
- Thiết kế web đơn giản bằng HTML có cộng đồng sử dụng đông đảo
- Dễ dàng tích hợp với các ngôn ngữ khác PHP, Node.js

Ưu, nhược điểm của HTML trong thiết kế website

Nhược điểm

- Không linh hoạt bởi HTML chỉ thực hiện theo logic, cấu trúc cố định
- Sử dụng chủ yếu trong web tĩnh
- Khó khăn trong việc kiểm soát các thao tác thực hiện của trình duyệt
- Chưa đủ các tính năng để thiết kế website Responsive hoàn thiện

3. Cách thức hoạt động của HTML

- Đuôi file HTML thường có dạng htm/.html
- Một website có nhiều HTML, mỗi trang con lại có những file HTML riêng
- File HTML thường bao gồm bộ tag (elements) có thể tạo được cấu trúc như các thư mục: section, heading, paragraph.
